le positionnement

div { position: static; pas besoin de top, left, right, bottom; }

static positionne le carré par rapport au contenu de la page. c'est la valeur par défaut

div { position: sticky; + top, left, right, bottom; }

sticky positionne le carré par rapport à l'écran. il ne bouge pas quand je scroll

div { position: absolute; top: 60%; left: 20%; }

absolute positionne le carré par rapport à la page

div { position: relative; + top, left, right, bottom; }

relative positionne le carré par rapport à son conteneur
