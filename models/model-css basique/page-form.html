les formulaires

les attributs name et value permettent de traiter les données du formulaire.

date, type='date' heure, type='time'

les champs date et heure sont difficilement stylisables.

type='radio' un seul name pour tous les éléments d'un même block. attribut checked afin de cocher une valeur par défaut. attribut required afin de rendre la saisie du champ obligatoire. attribut disabled afin d'empêcher le remplissage par l'utilisateur. un seul champ sélectionnable à la fois. type='hidden' afin de passer des infos à l'action sans prévenir l'utilisateur.
type='checkbox' mêmes infos que pour radio. plusieurs champs sélectionnables à la fois.

type='submit' ou reset. l'attribut value permet de choisir ce que le bouton affiche.

input:hover {background-color: #399; } input { border: solid 2px orange; outline: solid 4px #6F6; } outline est une sorte de border autour des input, button, select, textarea.
