les images et les liens

les attributs src et alt sont obligatoire. par défaut, laisser alt vide.

img { width: 400px; height: 200px; object-fit: fill; }

object-fit: fill. ajuster l'image à son cadre, en la déformant. affichage par défaut si width et heigth sont précisés.

object-fit: contain. ajuster l'image dans son cadre, en gardant ses proportions. elle peut être plus petite que son cadre.

img { object-fit: cover; object-position: left top; }

ajuster l'image à son cadre, en gardant ses proportions. elle est rognée si elle dépasse de son cadre.

lien

l'attribut href est obligatoire, même s'il est vide.

<base target='_blank' href='C:/Users/deborah.powers/Desktop/articles/'/>

base pré-écrit des infos pour les liens et les images. on le met dans le head.

valeurs de href: # désactiver le lien, #id un élément de la page qui a pour id id, http://www.truc.fr/ une page web, c://truc/machin.html un fichier (ou un dossier) local.
