______ utilisation à distance ______

créer un répertoire sous github
	faire le lien. origin est un nom par défaut, comme master en locale
	git remote add origin https://github.com/nom-github/dossier.git
	git remote add origin https://github.com/deborah-powers/dossier.git
vérifier que le lien a été fait. renvoi origin normalement
	git remote
déposer le contenu de la branche master dans origin
	git push origin master
récupérer un projet chez moi. git demande nom et mdp de github
	git pull origin master
	git push origin <local-branch-name>:<remote-branch-name>
me connecter chez quelqu'un d'autre
	git remote add nom-autre master adresse
aller sur une branche du remote
	git fetch origin
	git checkout mabranche
voir les infos du site distant
	git remote show origin
récupérer les données de distant/random à local/deborah
	les récupérer sur local/random
		git checkout random
		git pull origin random
	merger local/random dans local/deborah
		checkout deborah
		git merge random

Terminal=false

______ fonctions utiles ______

https://docs.mongodb.com/manual/mongo/
https://docs.mongodb.com/manual/reference/mongo-shell/

aide pour les bdd
	db.help()
voir la liste des bdd
	db.adminCommand( { listDatabases: 1 } )
	show databases
supprimer une bdd
	use dbToDelete
	db.db.dropDatabase()
aller dans une bdd
	use dbName
voir la liste des collections
	show collections
aide pour les collections
	db.collectionName.help()
créer une collection
	db.createCollection ('ColName', json_object)
voir le contenu d'une collection
	db.collectionName.find()
reperer un element dans une collection
	db.collectionName.findOne ({ field: 'value' })
supprimer un element d'une collection
	db.collectionName.remove ({ field: 'value' }, true)
